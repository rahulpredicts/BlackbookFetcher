Credential Configuration

Current Configuration

Credentials are stored securely in the .env file on the server.

To update credentials, modify the .env file with the following variables:

  • BLACKBOOK_ID - Your Blackbook account ID
  • BLACKBOOK_PASSWORD - Your Blackbook password
  • BLACKBOOK_GRAPHQL_URL - Blackbook GraphQL endpoint URL

Never store credentials in browser localStorage or expose them in the frontend.
